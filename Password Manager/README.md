# Password Manager (Python)

## Overview

This is a beginner-friendly Password Manager built using Python. It allows users to securely store and retrieve website login credentials through a simple command-line interface.

This project was created to practice Python programming concepts such as functions, dictionaries, loops, exception handling, and file handling.

---

## Features

- Add a new password
- Retrieve a saved password
- Store credentials in a local file
- Handle missing files safely
- Simple menu-driven interface

---

## Technologies Used

- Python 3
- Dictionaries
- File Handling
- Exception Handling

---

## How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run:

```bash
python passwordManager.py
```
